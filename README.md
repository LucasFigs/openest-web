# Openest Web

Frontend Web da plataforma Openest, desenvolvido com React, Vite e TailwindCSS.

## 🚀 Tecnologias

- React 18
- Vite
- TailwindCSS
- React Router DOM
- Axios
- Context API

## 📋 Pré-requisitos

- Node.js 18.x ou superior
- NPM ou Yarn

## 🔧 Instalação

1. Clone o repositório

```bash
git clone https://github.com/openest/openest-web.git
cd openest-web
```
